## What is Freddy's Kubernetes Helper (Fkh)

Fkh lets authorised GitHub users provision containers and many other developer tasks
on demand directly from VS Code.

A GitHub-authenticated Azure Function acts as the provisioning gate;

Terraform manages all Azure and GitHub infrastructure.
